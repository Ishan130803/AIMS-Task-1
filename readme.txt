# This is the Submission for AIMS task - 1 which is descibed as follows:

Write a program for any 2 Imputers and 2 Encoders in python using Numpy and Pandas Library only

Submission file Descriptions:
1. OrdinalEncoder.ipynb : This program implements Ordinal Encoding. To perform Ordinal encoding , user needs to create encoder object and then, provide the pandas dataframe and the columns user wants to encoder. It returns a numpy array with encoded values

2. OneHotEncoder: This program implements One Hot Encoding

3. SimpleImputer : It is a package of simple imputations as follows:
   - Constant Imputation
   - Mean Imputation
   - Median Imputation
   - Most Frequent or Modal imputation

  It is compatible with both numerica and categorical values
